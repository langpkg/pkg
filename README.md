- ## Quick Start 🔥

    ```bash
    # install via bun
    bun i @langpkg/pkg -g

    # Now use 'pkg' anywhere
    pkg --help
    ```

    > **This is not the final version (I’m not providing any guarantees here).**
    >
    > **I may change anything at any time based on my personal needs (sometimes I forget to update the docs :P)**
    >
    > **This tool and these rules were created specifically for my personal use (Fixing my `OCPD` and `ADHD`).**
